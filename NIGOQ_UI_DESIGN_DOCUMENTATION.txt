NIGOQ 模組 - UI與視覺設計完整文檔
=====================================

模組基本資訊
============
- 模組名稱: NIGOQ
- 尺寸: 12HP (180 x 380 像素)
- 背景色: 灰色 (#f0f0f0)
- 功能: 複雜合成器模組，具有雙振盪器、濾波器、調製等功能

標題區域設計
============
主標題:
- 文字: "N I G O Q" (字母間有空格)
- 位置: Vec(0, 1) 寬度180
- 字體大小: 12pt
- 顏色: 金黃色 RGB(255, 200, 0)
- 樣式: 粗體

副標題:
- 文字: "MADZINE"
- 位置: Vec(0, 13) 寬度180
- 字體大小: 10pt
- 顏色: 金黃色 RGB(255, 200, 0)
- 樣式: 一般

特殊數字標識
============
大型數字 "259" (品牌標識):
- 位置: Y=248 橫向排列
- "2": Vec(20, 248) 尺寸30x35
- "5": Vec(45, 248) 尺寸30x35
- "9": Vec(70, 248) 尺寸30x35
- 字體大小: 72pt
- 文字顏色: 白色 RGB(255, 255, 255)
- 邊框顏色: 黑色 RGB(0, 0, 0)
- 風格: 帶黑色描邊的白色數字

旋鈕設計規格
============

1. 大型白色旋鈕 (LargeWhiteKnob) - 37x37像素
   特徵:
   - 外圈: 深灰色 RGB(30, 30, 30)
   - 外圈邊框: 中灰色 RGB(100, 100, 100)
   - 內圈: 純白色 RGB(255, 255, 255)
   - 指示器: 粉紅色線條和點 RGB(255, 133, 133)
   - 旋轉範圍: -135° 至 +135°

   使用位置:
   - MOD FREQ: Vec(55, 65) - 調製頻率
   - PRIN FREQ: Vec(125, 65) - 主頻率

2. 標準黑色旋鈕 (StandardBlackKnob) - 38x38像素
   使用VCV Rack標準黑色旋鈕樣式

   使用位置:
   - LPF CUTOFF: Vec(125, 130) - 低通濾波器截止頻率
   - ORDER: Vec(125, 175) - 階數控制
   - HARMONICS: Vec(125, 220) - 諧波控制

3. 小型白色旋鈕 (SmallWhiteKnob) - 23x23像素
   特徵:
   - 外圈: 純白色 RGB(255, 255, 255)
   - 外圈邊框: 黑色 RGB(0, 0, 0)
   - 內圈: 淺灰色 RGB(240, 240, 240)
   - 指示器: 粉紅色線條和點 RGB(255, 133, 133)

   使用位置 (CV衰減器):
   - MOD WAVE: Vec(20, 55) - 調製波形選擇
   - FM_AMT_ATTEN: Vec(55, 130) - FM量衰減
   - FOLD_AMT_ATTEN: Vec(55, 220) - 折疊量衰減
   - AM_AMT_ATTEN: Vec(55, 175) - AM量衰減
   - MOD_FM_ATTEN: Vec(77, 310) - 調製FM衰減
   - PRIN_FM_ATTEN: Vec(108, 310) - 主FM衰減

4. 小型灰色旋鈕 (SmallGrayKnob) - 28x28像素
   特徵:
   - 外圈: 深灰色 RGB(30, 30, 30)
   - 外圈邊框: 中灰色 RGB(100, 100, 100)
   - 內圈: 純白色 RGB(255, 255, 255)
   - 指示器: 粉紅色線條和點 RGB(255, 133, 133)

   使用位置 (主要參數):
   - DECAY: Vec(165, 90) - 衰減時間
   - BASS: Vec(165, 265) - 低頻增強
   - FM_AMT: Vec(90, 130) - FM調製量
   - FOLD_AMT: Vec(90, 220) - 波形折疊量
   - AM_AMT: Vec(90, 175) - AM調製量

開關設計
========
三段開關 (CKSSThree):
- SYNC_MODE: Vec(90, 85) - 同步模式選擇

輸入/輸出插孔
=============
使用標準 PJ301MPort 插孔 (白色圓圈)

輸入插孔:
- TRIG_IN: Vec(165, 55) - 觸發輸入
- MOD_WAVE_CV: Vec(20, 95) - 波形CV
- LPF_CUTOFF_CV: Vec(165, 130) - LPF CV
- ORDER_CV: Vec(165, 175) - 階數CV
- FM_AMT_CV: Vec(20, 130) - FM量CV
- HARMONICS_CV: Vec(165, 220) - 諧波CV
- FOLD_AMT_CV: Vec(20, 220) - 折疊量CV
- AM_AMT_CV: Vec(20, 175) - AM量CV
- MOD_FM_IN: Vec(50, 310) - 調製FM輸入
- MOD_1VOCT: Vec(20, 310) - 調製1V/Oct
- PRIN_FM_IN: Vec(135, 310) - 主FM輸入
- PRIN_1VOCT: Vec(165, 310) - 主1V/Oct

輸出插孔:
- MOD_SIGNAL_OUT: Vec(20, 360) - 調製信號輸出
- PRIN_SINE_OUT: Vec(135, 360) - 主正弦波輸出
- PRIN_FINAL_OUT: Vec(165, 360) - 最終輸出

文字標籤設計
============
所有標籤使用白色 RGB(255, 255, 255)，字體大小7pt
輸出標籤使用粉紅色 RGB(255, 133, 133)

輸入標籤:
- TRIG, WAVE, LPF, ORD, FM, HARM, FOLD, AM
- M.FM, M.V/O, P.FM, P.V/O

參數標籤:
- MOD FREQ, PRIN FREQ
- LPF, ORD, FOLD
- WAVE, CV ATT (多個)
- DEC, BASS, FM, FOLD, AM, SYNC

輸出標籤 (粉紅色):
- MOD, SINE, FINAL

特殊UI元件
==========

1. 視覺顯示區域 (VisualDisplay)
   - 位置: Vec(40, 335)
   - 尺寸: 66 x 38.5 像素
   - 背景: 白色框架 RGB(255, 255, 255)
   - 邊框: 淺灰色 RGB(200, 200, 200)
   - 功能: 顯示波形示波器
   - 紋理解析度: 1024x768 (內部渲染)

2. 隱藏時間控制旋鈕 (HiddenTimeKnob)
   - 位置: Vec(40, 335) (與視覺顯示重疊)
   - 尺寸: 66 x 38.5 像素
   - 完全透明，僅響應垂直拖動
   - 鼠標懸停時顯示垂直調整游標

3. 示波器觸發按鈕 (ClickableLight)
   - 位置: Vec(110, 330) (視覺顯示右側)
   - 尺寸: 8 x 8 像素
   - 外觀: 圓形燈光按鈕
   - 顏色狀態:
     * 觸發禁用(自由運行): 亮粉紅色 RGB(255, 133, 133)
     * 觸發啟用: 暗粉紅色 RGB(200, 100, 100)
   - 邊框: 淺灰色 RGB(200, 200, 200) 0.5pt
   - 功能: 點擊切換示波器觸發模式開/關

視覺層級
========
1. 背景層: 灰色背景
2. 白色背景框: Y=330 位置的白色區域
3. 數字標識層: "259" 大型數字
4. 控制元件層: 旋鈕、開關、插孔
5. 文字標籤層: 所有文字標籤
6. 視覺顯示層: 示波器顯示

顏色方案總結
============
- 主背景: #f0f0f0 (淺灰)
- 標題文字: RGB(255, 200, 0) (金黃)
- 一般標籤: RGB(255, 255, 255) (白色)
- 輸出標籤: RGB(255, 133, 133) (粉紅)
- 旋鈕指示器: RGB(255, 133, 133) (粉紅)
- 深灰色: RGB(30, 30, 30)
- 中灰色: RGB(100, 100, 100)
- 淺灰色: RGB(240, 240, 240)

右鍵菜單選項
============
1. Oversampling Rate (過採樣率)
   - x1, x2, x4, x8, x16 選項

2. Randomize Glide Time (隨機滑音時間)
   - Off, Fast, Medium, Slow 選項

3. Random Amount (隨機量)
   - Off, Low, Medium, High 選項

交互設計特點
============
1. 所有旋鈕支持精確的鼠標拖動控制
2. 大型白色旋鈕使用X-Y混合拖動 (水平和垂直結合)
3. 小型旋鈕主要使用垂直拖動
4. 隱藏時間控制提供直觀的示波器時間調整
5. 粉紅色指示器提供清晰的視覺反饋

模組整體設計理念
================
- 簡潔現代的界面設計
- 清晰的功能分區
- 獨特的"259"品牌標識
- 粉紅色作為強調色，增加視覺辨識度
- 白色和灰色的對比創造層次感
- 緊湊但不擁擠的佈局
