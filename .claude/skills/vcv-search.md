# /vcv-search

搜尋 VCV Rack 和 DSP 開源資源

## 使用方式
```
/vcv-search [搜尋主題]
```

## 範例
```
/vcv-search "reverb algorithm"
/vcv-search "euclidean rhythm implementation"
/vcv-search "VCV polyphonic cable handling"
/vcv-search "oscillator anti-aliasing"
```

## 搜尋範圍
1. **VCV 社群**: community.vcvrack.com
2. **GitHub**: VCV Rack 相關專案
3. **DSP 資源**: musicdsp.org, ccrma.stanford.edu
4. **學術論文**: 音樂資訊檢索相關

## 輸出
- 相關開源專案列表
- 授權相容性檢查 (vs GPL-3.0)
- 程式碼品質評估
- 整合建議
- 關鍵程式碼摘錄

## 授權篩選
自動過濾不相容授權:
✅ GPL-3.0, MIT, BSD, Apache-2.0
❌ 專有授權, GPL-2.0-only
